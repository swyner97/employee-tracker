# employee-tracker

###  A command-line application where the user can manage a company's employee database, using Node.js, Inquirer and MySQL.

 ------------------

## TECHNOLOGIES USED:

- Node.js
- MySQL
- Javascript 

 ------------------

## DEPENDENCIES:

- MySQL2 
- Inquirer
- Connection

 ------------------

## INSTALLATION:

- In integrated terminal:

        npm i mysql2
        npm i node
        npm i connection

- In MySQL terminal: 

        -u root -p 
        source db/schema.sql
        source db/seeds.sql

- In integrated terminal:
            
        node index.js
