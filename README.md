# employee-tracker

###  A command-line application from scratch to manage a company's employee database.

 ------------------

## TECHNOLOGIES USED:

- Node.js
- MySQL
- Javascript 

 ------------------

## DEPENDENCIES:

- MySQL2 
- Inquirer
- Connection

 ------------------

## INSTALLATION:

- In integrated terminal:

        npm i mysql2
        npm i node
        npm i connection

- In MySQL terminal: 

        -u root -p 
        source db/schema.sql
        source db/seeds.sql

- In integrated terminal:
            
        node index.js
